Help with creating a new role

An role allows grouping of domains belonging to a particular entity, such as a customer or department.
A domain can be assigned to an role upon domain creation or through the domain administration page.

Fill in all the fields to the in the form to the left.

Name is an role identifier. It will be stored as all lowercase letters (no spaces, special characters etc).
Description is a user friendly name for this role.
Contact person is the name of a contact person at the role.
Mail Address is an e-mail address for the contact person.
